# MoviesApp  :family:
## *Overview
> This app applied Simple CRUD Operations (Create, Read, Update, Delete) Movie and the app also 
support multi-languages(English - Arabic) 

## Tools
- Asp.net core 5.0
- EF core
- Sql Server 
- bootbox.js
- animate.css
- bootstrap-datepicker
